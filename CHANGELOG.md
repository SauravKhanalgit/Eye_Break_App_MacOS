# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-11-29

### Added
- Initial release of Eye Break App
- Menu bar integration for macOS
- Countdown timer display in menu bar
- System notifications for break reminders
- Configurable break intervals (20, 30, 60 minutes)
- Pause/Resume timer functionality
- Manual break trigger
- Timer reset functionality
- 20-20-20 rule reminder system
- Main application window with visual timer
- Status indicators (Running/Paused)

### Features
- Runs as a background menu bar app
- Minimal resource usage
- Clean and intuitive user interface
- Persistent timer across app restarts
